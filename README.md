# SOCKS5 Auto Installer

This script installs and configures a SOCKS5 server (dante-server) on Ubuntu.

## Usage

**One-line command (after you push this to your repo):**

```bash
wget -O - https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/install_socks5.sh | bash -s -- <username> <password> <port>
```

Or interactively:

```bash
wget -O - https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/install_socks5.sh | bash
```

## Manual Usage

1. Clone this repository.
2. Make the script executable: `chmod +x install_socks5.sh`
3. Run it: `./install_socks5.sh [username] [password] [port]`
